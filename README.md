# Canteen Management System 🍽️

A comprehensive **Canteen Management System** built to automate and efficiently manage the daily operations of college, office, or institutional canteens. It streamlines menu management, order processing, inventory tracking, billing, and reporting.

## Features

### **Core Features**
- **Multi-User Role System** (Admin, Canteen Staff, Customer)
- **Secure Authentication & Authorization**
- **Dynamic Menu Management** (Add/Edit/Delete food items with images & pricing)
- **Online & Counter Order Placement**
- **Real-time Order Tracking** (Pending → Preparing → Completed → Delivered)
- **Inventory & Stock Management** with low-stock alerts
- **Billing & Invoice Generation**
- **Sales Analytics & Reports**
- **Responsive Web Design**

### **Admin Dashboard**
- Overview of today's sales and orders
- Manage users, menu, and inventory
- Generate daily/weekly/monthly reports

### **Customer Features**
- Browse menu with categories
- Add items to cart
- Place orders easily
- View order history
